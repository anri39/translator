# Language Translator App

A simple React-based language translator using the DeepL API.

## Features

- Real-time text translation between multiple languages
- Auto-detect source language option
- Character count and input validation

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Get a free DeepL API key from [https://www.deepl.com/pro-api](https://www.deepl.com/pro-api)
4. Open `src/App.tsx` and replace the `apiKey` variable with your actual API key:
   ```typescript
   let apiKey = "your-actual-deepl-api-key-here";
   ```
5. Run the development server: `npm run dev`

## Usage

1. Enter text in the input box
2. Select source and target languages
3. Press Enter
4. View the translated text in the output box

## Technologies Used

- React + TypeScript
- Vite
- DeepL API
